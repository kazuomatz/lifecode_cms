# README

## Product Name

LifeCode CMS for Rails 6.0

## Ruby version

2.6.5

## System dependencies

- Rails 6.0

- MySQL 6.9

- Mecab

- ImageMagick 5.7

## Getting Started
```
$ bundle
$ yarn install
$ rake db:create
$ rake db:migrate
$ rake db:seed
```
