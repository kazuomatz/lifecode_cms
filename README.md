# README

## Product Name

LifeCode CMS for Rails 6.0

## Ruby version

2.6.5

## System dependencies

- Rails 6.0

### RDBMS
- MySQL 8.0

### Requirements for Ruby Libraries
- Mecab
- mecab-ipadic-NEologd
- ImageMagick 6.9x

### Requirements for Frontend
- jQuery 3.x
- Bootstrap 4.x
- coreUI 2.0x (https://coreui.io/)
- parsley.js (https://parsleyjs.org/)
- iCheck(https://github.com/fronteed/icheck)
- Fontawesome 5x Free (https://fontawesome.com/)
- bootstrap-datepicker, bootstrap-timepicker 

## Getting Started
```
$ bundle
$ yarn install
$ rake db:create
$ rake db:migrate
$ rake db:seed
```
